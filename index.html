Curso Gratuito

Curso de HTML5 e CSS3

Neste curso você aprenderá as principais tags e elementos do HTML5 para estruturar conteúdo de forma semântica. Veremos como utilizar o CSS3 para estilizar e dar formatação visual aos elementos HTML através de seletores, propriedades e valores. Aprenderemos também sobre layouts responsivos para que o conteúdo se adapte a diferentes dispositivos. Ao final do curso você saberá criar páginas web modernas com HTML5 e CSS3.

Curso de JavaScript

Neste curso básico de JavaScript você aprenderá os fundamentos da linguagem para adicionar interatividade e comportamento dinâmico às páginas web. Veremos como incluir scripts JavaScript nos arquivos HTML, variáveis, tipos de dados, operadores, estruturas de controle como condicionais e repetição. Aprenderemos também sobre funções, métodos built-in, manipulação de elementos DOM e eventos. Como projeto final criaremos um pequeno jogo ou aplicação interativa para consolidar os conhecimentos adquiridos ao longo das aulas. Ao finalizar este curso você estará apto a desenvolver sites e aplicações web dinâmicas com JavaScript.
